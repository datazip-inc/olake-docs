# OLake - The Open Lakehouse Platform

## Company Overview
OLake is a cutting-edge open-source data lakehouse platform that provides the fastest way to replicate MongoDB data into Apache Iceberg.

## Core Product
OLake - A high-performance data replication platform that enables:
- MongoDB to Apache Iceberg replication
- Real-time CDC capabilities
- Support for multiple databases (MySQL, PostgreSQL, Oracle)
- Integration with Apache Iceberg, Delta Lake, and other table formats
