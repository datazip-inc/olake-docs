
A typical `state.json` file has the following structure:

```json title="state.json"
{
    "type": "STREAM",
    "streams": [
        {
            "stream": "sample_table",
            "namespace": "main",
            "sync_mode": "",
            "state": {
                "binlog_file": "mysql-bin.000003",
                "binlog_position": 1027,
                "chunks": [],
                "server_id": 1000
            }
        }
    ]
}
```
