
A typical `state.json` file has the following structure:

```json title="state.json"
{
    "type": "GLOBAL",
    "global": {
        "state": {
            "lsn": "0/198C935"
        },
        "streams": [
            "public.sample_data"
        ]
    },
    "streams": [
        {
            "stream": "sample_data",
            "namespace": "public",
            "sync_mode": "",
            "state": {
                "chunks": []
            }
        }
    ]
}
```
