| Permission / Privilege      | Use                                                                                                                | Where it's Used                                                                              | Query to Grant                                                 | Error Code Without Permission                                             |
|-----------------------------|--------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------|----------------------------------------------------------------|---------------------------------------------------------------------------|
| `CREATE SESSION`            | Fundamental system privilege that allows a user to log in (connect) to the Oracle database.                        | To connect to the database (DBeaver, SQL Developer, Olake, etc.).                            | `GRANT CREATE SESSION TO <username>;`                          | `ORA-01045: user <username> lacks CREATE SESSION privilege; logon denied` |
| `EXECUTE ON DBMS_FLASHBACK` | Oracle-supplied PL/SQL package that allows users to "flash back" their session to a previous point in time or SCN. | Used to fetch the SCN for the table.                                                         | `GRANT EXECUTE ON DBMS_FLASHBACK TO <username>;`               | `ORA-00904: "SYS"."DBMS_FLASHBACK": invalid identifier`                   |
| `SELECT ON <table>`         | Grants read-only access to run SELECT queries on the table owned by the schema owner.                              | Required to access the table for discovery and chunk iteration.                              | `GRANT SELECT ON <table_owner>.<table_name> TO <username>;`    | `ORA-00942: table or view does not exist`                                 |
| `FLASHBACK ON <table>`      | Grants the FLASHBACK object privilege for the table. Allows querying historical data using AS OF SCN.              | Required when running queries like SELECT * FROM table AS OF SCN ... during chunk iteration. | `GRANT FLASHBACK ON <table_owner>.<table_name> TO <username>;` | `ORA-01031: insufficient privileges`                                      |
