| **Key**                  | **Description**                     | **Sample Value**                |
|--------------------------|-------------------------------------------------------------------------------------|-------------------------------------------------------------------------|
| **S3 Bucket**     | Name of the object storage bucket where Parquet files are stored. This can be any S3 compatible object storage bucket. The bucket must already exist and be writable.                                                                                                                     | `warehouse`                        |
| **S3 Region**     | AWS Region where your S3/GCS bucket exist. Required so OLake Go calls the correct regional S3 endpoints.                                                                                                                                                                                                     | `ap-south-1`            |
| **S3 Access Key** | Access key for authenticating S3 API requests. Required for static credentials (AWS, GCS HMAC, or MinIO). Optional when using an IAM role.                                                                                                                                            | `XXX`                                                 |
| **S3 Secret Key** | The AWS/MinIO/GCS HMAC Secret key paired with the access key. Required with S3 Access Key for static credentials. Optional when using an IAM role.                                                                                                                                | `XXX`                                                 |
| **S3 Path**       | Folder prefix inside the bucket where Parquet files are written. If left empty, files go to the bucket root.                                                                                          | `/data`                                                     |
| **S3 Endpoint**    | S3 API endpoint for writing parquet files. Optional for AWS S3 but required for S3-compatible storage like MinIO.                                                          | `https://glue.ap-south-1.amazonaws.com`                 |
