---
title: "AWS S3 Destination Configuration - OLake Parquet Writer"
description: "Configure AWS S3 as destination for OLake Parquet writer. Set up S3 bucket, region, access keys for data lake storage."
---

```json title="destination.json"
{
    "type": "PARQUET",
    "writer": {
      "s3_bucket": "<BUCKET_NAME>",
      "s3_region": "ap-south-1",
      "s3_access_key": "XXX",
      "s3_secret_key": "XXX",
      "s3_path": "/data",
      "s3_endpoint": "http://<S3_ENDPOINT>:9000"
    }
}
```
