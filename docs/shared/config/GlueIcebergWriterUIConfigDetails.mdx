
| Parameter         | Sample Value                                   | Description                                                                                             |
|-------------------|------------------------------------------------|---------------------------------------------------------------------------------------------------------|
| **Iceberg S3 Path (Warehouse)**       | `s3://<BUCKET_NAME>/`      | S3 bucket path where Iceberg table data and metadata files will be stored.                            |
| **AWS Region**    | `ap-south-1`                                   | AWS region containing the S3 bucket and Glue Data Catalog resources.                                  |
| **AWS Access Key**| `XXX`                                          | AWS access key ID for authentication. Optional if using IAM roles or instance profiles.              |
| **AWS Secret Key**| `XXX`                                          | AWS secret access key for authentication. Optional if using IAM roles or instance profiles.          |
| **Iceberg Database**      | `olake_iceberg`                                | Database name to create in AWS Glue Data Catalog for organizing Iceberg tables.                      |
| **gRPC Port**     | `50051`                                        | Port number for the gRPC server to listen on for client connections. Optional, defaults to 50051.    |
| **Server Host**   | `localhost`                                    | Hostname or IP address where the gRPC server will bind and accept connections. Optional, defaults to localhost. |
