---
title: "Docker Sync PostgreSQL Command - OLake Docker"
description: "Execute OLake sync for PostgreSQL using Docker. Replicate PostgreSQL tables to data lakes with containerized OLake setup."
---

<Tabs groupId="docker">

<TabItem value="macos-linux" label="macOS / Linux" default>

```bash
docker run --pull=always  \
  -v "$HOME/PATH_TO_OLAKE_DIRECTORY:/mnt/config" \
  olakego/source-postgres:latest \
  sync \
  --config /mnt/config/source.json \
  --catalog /mnt/config/streams.json \
  --destination /mnt/config/destination.json
```
</TabItem>

<TabItem value="cmd" label="CMD">

```bash
docker run --pull=always  ^
  -v "%USERPROFILE%\PATH_TO_OLAKE_DIRECTORY:/mnt/config" ^
  olakego/source-postgres:latest ^
  sync ^
  --config /mnt/config/source.json ^
  --catalog /mnt/config/streams.json ^
  --destination /mnt/config/destination.json
```
</TabItem>

<TabItem value="powershell" label="Powershell">

```bash
docker run --pull=always  `
  -v "$env:USERPROFILE\PATH_TO_OLAKE_DIRECTORY:/mnt/config" `

  sync `
  --config /mnt/config/source.json `

  --destination /mnt/config/destination.json
```
