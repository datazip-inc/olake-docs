# `source.json`

The source configuration varies by database type. Click on the appropriate database tab below to access the detailed configuration guide:

<Tabs>

<TabItem value="mongo-config" label="MongoDB" default>

For detailed MongoDB source configuration, including connection parameters, authentication options, and advanced settings, please refer to the [MongoDB Source Configuration Guide](../../connectors/mongodb/config).

</TabItem>

<TabItem value="postgres-config" label="Postgres">

For detailed Postgres source configuration, including connection parameters, SSL settings, and replication options, please refer to the [Postgres Source Configuration Guide](../../connectors/postgres/config).

</TabItem>

<TabItem value="mysql-config" label="MySQL">

For detailed MySQL source configuration, including connection parameters, SSL settings, and binlog configuration, please refer to the [MySQL Source Configuration Guide](../../connectors/mysql/config).

</TabItem>

<TabItem value="oracle-config" label="Oracle">

For detailed Oracle source configuration, including connection parameters, authentication, and LogMiner setup, please refer to the [Oracle Source Configuration Guide](../../connectors/oracle/config).

</TabItem>

</Tabs>
