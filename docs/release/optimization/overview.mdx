# Release Notes

Welcome to the OLake Optimization release notes archive. Each release contains a changelog of new features, enhancements, and bug fixes. Use this landing page to:
- Quickly navigate to a specific version’s release notes
- Understand when major features were introduced
- Learn about backward-incompatible changes and deprecations

**Why a Release Landing Page?** OLake Optimization evolves rapidly with new table-maintenance capabilities, tuning improvements, and stability enhancements. Since not all optimization features are available in every version, this page helps you identify when key maintenance capabilities were introduced, track deprecations, and access guidance for migrating between versions.

### Navigate Releases

Use the Release dropdown in the documentation sidebar to select any version’s detailed notes. Each version page includes:
- 🎯 What's New: Additions or enhancements introduced in this release across table-maintenance workflows, optimization strategies, scheduling controls, and configuration options that improve OLake Optimization efficiency and reliability.
- 🔧 Bug Fixes & Stability: Improvements and refinements to existing functionality in this release; reliability, performance, and compatibility fixes.
